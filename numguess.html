    Number Guessing Game

    This project was was completed with the goal of implementing a MongoDB and routes for communicaiton with the server. This applicaiton takes in a users name then creates a game record within the database and populates it with a randome number. Then the user geusses a number from 1-100. The guess is relayed to the database where it is checked. The user is informed if they are too high or too low. If the user is correct the game ends and the highscore table can be viewed.

    The Highscore table is filtered to only the top ten players with the shortest guesses, and have won the game.
