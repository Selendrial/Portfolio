    Air BnB Clone

    This project was completed as a development exorcise using ASP.NET 8.0 and styled with Bootstrap 5.3.3 and msSQL. The entire project was developed by a team of 5 from scratch. this project focused on the development life cycle from concept to deployment.

    This project was a challenge to build a full stack aplicaiton from the gound up. The project was chosen by the team and meetings began with the client. The client was consulted regularly for guidance and to understand user requirements. Once the requirements were established, dvelopment with asp.Net 8.0 was started. The AirBnB clone uses Identities and Entity core framworks to scafold the program as well as using a Microsoft SQL database backend. The system allows for booking of a property by a customer, and managing of the rental by a owning host.
