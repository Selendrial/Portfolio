    Hangman Game

    THis program uses Mongo Express React Node (MERN) stack and implemented the display of imagaes using useEffects to transition based off the results if user guesses. The words are dynamically generated within the database. once the user guesses the data base us queried to see if the letter is withing the target word. If the leter is within the target word, the leter is displayed and the hangman is not updated. The number of guesses taken and the word are stored to be displayed after the game is completed. The challenge with this game was to implement the use of a graphical hangman, as well as solving the implementation of a library to dynamically generate words to play against. The useEffect and useStates were used to address various changing display elements.
