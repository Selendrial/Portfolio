    Banking Applicaiton

    The goal of this project was to implament sessions for secure login applications. This program uses Node and Express to communicate with the MongoDB. Sessions are created and passed to the server, and saved to the database. Once the user logs in they are able to perform actions on thier account. Each action that is performed is recorded in a legger that the user can view at anytime. This application include secure passwords practices by implementing sha-256 hashing and salting values that are saved to the database rather than raw password data.
